Στην εργασια 2 εχει υλοποιηθει η γραμματικη miniJava.Εχω υλοποιησει 3 Visitors : 

(1) DefCollectorVisitor.java : συγκεντρωνει ολες τις δηλωσεις ονοματων κλασεων,πεδιων,μεθοδων,παραμετρων 
μεθοδων, και τοπικων μεταβλητων της καθε μεθοδου και τα αποθηκευει ολα στα 4 Maps που εχω δημιουργησει 
και η μορφη της καθε εγγραφης του καθε Map φαινεται στην αρχη του αρχειου SymbolTables.java

(2) CheckClassVVisitor.java : ελεγχει αν τα αντικειμενα τυπου κλασης που εχουν δηλωθει μεσα στο αρχειο 
εισοδου (πεδια,μεθοδοι,παραμετροι μεθοδων,τοπικες μεταβλητες) υπαρχουν και εχουν δηλωθει.Ο ελεγχος γινεται
μεσω των Maps που δημιουργησε ο DefCollectorVisitor

(3) TypeCheckVisitor.java : ελεγχει αν ολες οι πραξεις,αναθεσεις,κλησεις μεθοδων,τυποι επιστροφης και οτι
αλλο περιλαμβανει η γραμματικη ειναι νομιμα ως προς τον τυπο.Ο ελεγχος γινεται μεσω των Maps που δημιουργησε
ο DefCollectorVisitor

Επιπλεον το προγραμμα περιλαμβανει και τα εξης αρχεια:
-Main.java
-SymbolTables.java
-Offsets.java(δεν προλαβα να το κανω να λειτουργει καλα αλλα υπαρχει και στη Main.java σχολιασμενο το κομματι
που την καλει και εκτυπωνει στο output τα offsets)
-Makefile

Για τη μεταγλωτισση χρησιμοποιειται το Makefile(περιεχει και την εντολη clean για τη διαγραφη των .class
αρχειων).Το προγραμμα τρεχει με την εντολη:

		java Main [inputfile]

οπου [inputfile] το αρχειο εισοδου(το προγραμμα δεχεται ενα αρχειο εισοδου τη φορα)

Το προγραμμα εγινε σε linux ubuntu,και τρεχει κανονικα εκτυπωνοντας την επιτυχια η το error που εμφανιστηκε 
στο αρχειο εισοδου.
